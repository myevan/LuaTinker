LuaTinker
=========

How to build
------------

#### Requirements

* cmake
* lua-5.1.x

#### OS X

    $ brew install cmake 
    $ brew inbstall lua51
    $ ./build.sh


#### CentOS

    $ sudo yum install cmake 
    $ sudo yum inbstall lua-devel
    $ ./build.sh

#### Windows

* Download & Install CMake <http://www.cmake.org/download/>
* Run CMake (cmake-gui) on Desktop
* Build Visual Studio Solution File
